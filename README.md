# Keerthika Vetri – Portfolio

This repository contains the personal portfolio website of **Keerthika Vetri**, an AI & Data Science student and part‑time Computer Science teacher. The site showcases profile information, skills, projects, memberships, and contact details.

## Tech Stack

- **HTML5** for structure
- **CSS3** (inline `<style>` block) for layout and styling
- **Google Fonts** (`Poppins`) for typography

## Structure

- **`index.html`** – Main landing page (can be this file or a separate resume page, depending on setup)
  - Header with name, role, and brief tagline
  - Intro section with profile photo and About Me text
  - Navigation buttons to other pages
  - Contact section with email and LinkedIn link
- **`skills.html`** – Skills overview (linked from the buttons)
- **`projects.html`** – List and details of projects
- **`membership.html`** – Memberships or associations
- **Assets**
  - `photo1.jpeg` – Profile image used on the landing page

## How to Run

1. Clone or download this repository.
2. Place all HTML files and `photo1.jpeg` in the same directory (or update paths as needed).
3. Open `index.html` (or the main HTML file) in any modern web browser.

No additional build steps or dependencies are required.

## Customization

- Update the **header text** (name, role, tagline) in the `<header>` section.
- Replace `photo1.jpeg` with your own image, keeping the same filename or updating the `src` attribute.
- Edit the **About Me** paragraph to reflect your latest experience or goals.
- Modify the links in the buttons section if you rename or reorganize the HTML pages.
- Update contact details (email and LinkedIn) in the Contact section.
